Pegar el JSON de la API

Copia la respuesta completa de la API (el array de nodos) y pégala aquí. Puedes hacerlo cada vez que quieras refrescar la calidad de los nodos: nunca se te borran ni repiten los que ya tienes en uso o bloqueados.

Traer nodos directo de la API de Mysterium

Consulta en vivo discovery.mysterium.network sin copiar/pegar nada. Si el navegador bloquea la petición (CORS), usa el botón "Abrir en pestaña" y pega el resultado en el panel de la izquierda — funciona siempre.

Los parámetros de país/servicio se le piden a la API, pero además esta app siempre vuelve a filtrar GB y calidad por su cuenta, así que aunque la API ignore un parámetro, el resultado que ves sigue siendo correcto.
